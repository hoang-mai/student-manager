# HV-04 - Đề xuất kết quả học tập

## Thông tin chung
- **Nhóm người dùng:** Học viên
- **Mã chức năng:** HV-04
- **Tên chức năng:** Đề xuất kết quả học tập

## Mô tả
Tạo yêu cầu thêm, cập nhật hoặc xóa điểm kèm file minh chứng.

## Module liên quan
- Grade Request Module
- File Storage Module
- Notification Module

## Luồng hoạt động chi tiết
1. Học viên chọn môn học và loại đề xuất (thêm/sửa/xóa).
2. Điền lý do và upload file minh chứng (ảnh, PDF).
3. Gửi yêu cầu về hệ thống, trạng thái ban đầu là "Chờ duyệt".
4. Thông báo tự động gửi đến chỉ huy liên quan.
